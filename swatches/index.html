Swatches

Designers often find inspiration from other designers, and the use of color by designers across the web is of particular interest to me. Later, I forget where this color inspiration came from or the exact hex color values that were used in the design. To aid my ailing color memory and to improve my knowledge of color on the web, Swatches is a log containing color swatches of hex color values I don't want to forget and attributed if appropriate.
